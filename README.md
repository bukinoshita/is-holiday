## Install
```bash
$ npm install -g is-holiday
```

## Usage
```bash
$ is-holiday --help

  Usage:
    $ is-holiday          Check if today is a holiday

  Example:
    $ is-holiday
    $ is-holiday --br

  Options:
    --br, --brazil       Check if today is a holiday in Brazil
    -h, --help           Show help options
    -v, --version        Show version
```
